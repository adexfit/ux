PLENTYZ ENTERPRISE

  1. Home
  2. Plentyz Enterprise

Project scope and rationale

This project is an e-commerce website that was designed for a start-up here in Nigeria. The project lasted for one month. I was the only designer who worked on this project.

The rationale behind the project is to increase brand awareness and generate more sales for the company in Nigeria. Plentyz had been generating a good income from offline sales of products in Lagos but decided to become a major contender in the digital space by creating an e-commerce website that would increase sales and brand awareness.


Users and audience

The targeted audience for this project are users aged 18 - 50 years with expendable income.

My roles and responsibilities

  • Wireframing
  • Prototyping
  • User testing

The process

This project started with a brief from Plentyz which included all the necessary information like the name of the company, about the company, project goals, Key dates & deadlines, audience for project and deliverable outcomes e.t.c.

From the brief, I was able to extract the product information, business goals, and information about the targeted audience, which helped in the creation of a user persona.

After going through the user requirements and user pain points highlighted, I ideated on the possible means of solving the problem and narrow down my solution using scenarios. I created the wireframe on Figma which was reviewed by the client and was later developed into a high fidelity prototype on Figma.

User testing

I conducted a test on the design by recruiting participants who had the features outlined in the user persona for the project. All the users were able to easily navigate through the prototype.
